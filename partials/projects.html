PackStar

In ths App the media sources are checked to see which cameras are avaliable. If the rear camera is present, it will default to "environment". This functionality is included for android devices. A user can directly take a photo and have it update to a carousel. The image is drawn to a Canvas element then converted to a blob for storage and retrieval.

The user can also save customized lists and all images for future retrieval. A counter is in place for all items showing how much has been packed. All lists and titles can be edited and deleted.

This Application was inspired by the Movie Clueless. I've always wanted a virtual closet and now I have one that I can travel with.
This is an Angular app that I used Materialize and Angular-Material with. Canvas was used for the webcam streaming and image capture.
The image is converted from a string of pixels to a blob object then stored with Firebase storage. Responsive web design was used
so the App can be used on desktop and android devices.
